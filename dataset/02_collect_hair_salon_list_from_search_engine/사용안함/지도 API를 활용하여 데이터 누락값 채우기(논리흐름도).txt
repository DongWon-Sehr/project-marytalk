2020.04.01.
지도 API를 활용하여 데이터 누락값 채우기(논리흐름도)
서동원

변수
	df_all 		: 기존 미용실 정보 저장 DF
	df_all_c	: df_all 복제본 DF (reindex)
	df_modified	: 처리결과 저장 DF

논리흐름

	1. df_all_c = df_all.reindex(columns = ['업소명','전화번호','도로명주소','지번주소','업태명'])
		필요한 컬럼만 재색인

	2. df_all_c = df_all_c.reset_index()
		지역명이 담긴 인덱스를 컬럼으로 이동

	3. df_modified 새 Df 생성
		컬럼은 ['업소명','처리전 업소명','전화번호','도로명주소','지번주소','업태명','처리상태']
		
	4. 이제 df_all_c 데이터를 가지고 처리를 할 것

	5. n번째 row 데이터의 '업소명'으로 지도 API에 검색

	6. 처리
		(case 1) 검색결과 없음
			(1) n번째 데이터를 modified에 저장 ('처리상태'='0')
				-> n번째 데이터 df_all_c에서 삭제
			
		(case 2) 검색결과 1개
			(1) df_all_c에서 '주소'값이 검색결과의 '주소'값과 같은 데이터만 조회
			(2-1) 해당하는 모든 데이터를 df_modified에 저장 ('처리상태'='1')
				-> 해당 데이터 모두 df_all_c에서 삭제
			
			(2-2) 해당하는 데이터가 없다면 n번째 데이터를 modified에 저장 ('처리상태'='0')
				-> n번째 데이터 df_all_c에서 삭제
			
		(case 3) 검색결과 2개 이상
			(1) 검색결과를 모두 가져온다.
			(2) 검색결과에서 하나씩 차례로 가져와
				df_all_c에서 '주소'값이 일치하는 데이터만 조회
			(3) 해당하는 모든 데이터를 df_modified에 저장 ('처리상태'='1')
				-> 해당 데이터 모두 df_all_c에서 삭제
			(4-1) 검색결과 모두 차례로 수행
			
			(4-2) df_all_c에 검색결과의 모든 '주소'값과 일치하는 데이터가 아예 하나도 없다면 
				n번째 데이터를 modified에 저장 ('처리상태'='0')
				-> n번째 데이터 df_all_c에서 삭제
			
	7. df_all_c == False가 될 떄 까지 "6. 처리" 과정 반복
		df_all_c의 데이터가 모두 삭제될 떄 까지 반복

	8. df_modified.drop_duplicates(inplace=True)
		df_modified의 중복 데이터 제거
