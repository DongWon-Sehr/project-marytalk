네이버 지도 API를 활용하여 공공데이터 누락값 채우기

1.	활용 API

	네이버 개발자센터 - 지역 검색 API
	https://developers.naver.com/docs/search/local/


2.	네이버 지도에 검색한 결과 데이터를 받아볼 수 있음

	일일 최대 호출량			: 25,000
	1회 호출시 요청 가능 건수		: default 10건 / 최대 30건 조회
	1회 호출시 start point		: default 1 / 최대 1,000

	
3.	문제점
	
	일일 최대 호출량은 2만 5천건으로 미용실 공공 데이터를 처리하는데 문제가 없으나
	요청 가능 건수와 start point의 문제로 1,030건 이후의 데이터는 받아볼 수 없다.
	따라서 검색 결과가 1,030건이 넘지 않게 조절해야한다.
	
	원래 생각해본 방안은
	"서울 미용실"로 네이버 지도에 검색시 약 2만건의 결과가 나온다.
	미용실 공공데이터가 약 1만9천건 이므로
	얼추 데이터가 비슷하다고 판단.
	"서울 미용실"로 검색하여 정보를 모두 긁어와 공공데이터의 누락값을 한번에 채우려고 했으나
	1,030건 이후 1,031번째 데이터 ~ 마지막 데이터는 아예 조회 자체가 불가능.
	
	따라서 어떤식으로 검색하여 1,030건 이하의 검색결과가 나오게 할지 고민 해야봐야 한다.
